# Slider

Slider is a carousel of given elements. It's build with slick https://kenwheeler.github.io/slick/.

```js
import Slider from '@pagerland/common/src/components/Slider';

<Slider>
  <div>Slide A</div>
  <div>Slide B</div>
  <div>Slide C</div>
</Slider>;
```
